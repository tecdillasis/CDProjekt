Audio Visualizer

Salah satu elemen multimedia adalah audio. Penyajian dan pengambilan audio melalui aplikasi web dapat dikembangkan menggunakan canvas HTML. Melalui canvas HTML, data audio dapat diperoleh kemudian digunakan sebagai variabel input untuk memanipulasi elemen grafis yang dianimasikan (animated graphic element).
